IF ELSE / OPERADORES LÓGICOS - OR ||

if (condición1 && condición2 && condición3) {
//acciones a realizar en caso positivo
//...
} else {
//acciones a realizar en caso negativo
//...
}

Los operadores lógicos son muy parecidos a los relacionales, en el sentido de que dan también como output sólo dos valores que, en este caso, son:verdadera y falsa.

Operador Símbolo Significado
AND && devuelve true si se cumplen todas la condiciones
OR || devuelve true se si cumple una de las condiciones
