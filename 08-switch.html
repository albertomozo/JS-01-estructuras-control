Condición Switch

La sentencia switch se utiliza para realizar diferentes acciones basadas en diferentes condiciones.

Se utiliza la sentencia para seleccionar uno de los muchos bloques de código a ejecutar.

Así es como funciona:

  • La condición se evalúa una vez.
  • El valor de la condición se compara con los valores de cada caso.
  • Si hay una coincidencia, se ejecuta el bloque asociado al código.

Sintaxis:

switch(valor) {
case "texto" : instrucción1;
break;
case 12 :instrucción2;
break;
case "texto" :instrucción3;
break;
default:default instrucción
}
